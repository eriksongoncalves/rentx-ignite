## Configurações

<br />

Instale o json-server

```
yarn add -g json-server
```

<br />

Na raiz do projeto instale as dependências:

```
yarn install
```

<br />

## Iniciando a aplicação:

<br />

Altere o arquivo `./src/services/api.ts` com o seu IP.

<br />

**Servidor**

```
yarn api
```

**Aplicação**

```
expo start -c
```
